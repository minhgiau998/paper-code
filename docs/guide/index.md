# Getting Started

Welcome to PAPER-CODE! Start here to learn how to generate AI-ready documentation for your projects.

## What is PAPER-CODE?

PAPER-CODE is an AI-Native Documentation Generator that automatically creates:

- **AI Context Files** (`.cursorrules`, `AI_RULES.md`)
- **Architecture Guides** (stack-specific best practices)
- **Governance Files** (LICENSE, CONTRIBUTING, SECURITY, etc.)
- **Library Documentation** (100+ supported libraries)

All optimized for Cursor, Windsurf, and Copilot.
